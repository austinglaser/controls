How to use these files:
1) Open torsional-interface.lvproj and compile the torsional-fpga-interface.lvbit
2) Copy the newly created bitfile to C:\Program Files\National Instruments\LabVIEW 8.20\user.lib\torsional-fpga-interface.lvbit and make it read-only
3) Copy torsional-error-handler.vi to c:\Program Files\National Instruments\LabVIEW 8.20\user.lib\ and make it read-only
4) Copy torsional-interface-template.vit and torsional-interface-templated.png to C:\Program Files\National Instruments\LabVIEW 8.20\Installation
5) Open the new copy of torsional-interface-template.vit.  
6) Save torsional-interface-template.vit and make it read-only.
7) Verify that you can use the template in LabVIEW using File->New from Template...
